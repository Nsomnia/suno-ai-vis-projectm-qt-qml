## [0.0.1] - 2024-01-XX

### Added
- Initial project structure and CMake configuration
- CPM.cmake dependency management integration
- Core configuration management via TOML (ConfigManager)
- Git version control wrapper (GitController) using libgit2
- Qt6/QML application bootstrap with modern accordion UI
- Theme system foundation (Light/Dark mode support)
- libprojectM v4.1.x integration scaffolding
- FFmpeg video capture pipeline foundation
- Suno API client skeleton
- Audio processing pipeline architecture (stem isolation prep)
- Main canvas area for visualizer rendering
- Side drawer accordion components
- Settings window infrastructure

### Technical
- C++23 standard adoption
- Qt 6.6+ QML interface
- Strict SRP (Single Responsibility Principle) enforcement
- Home directory config storage (~/.config/suno-visualizer/)
- Zero monolithic dependencies in proprietary repo
